void Ball::handleCollision ( Collision  Collision)
virtual

When a collision detects, this method changes status of this ball. First, detach the ball with the collided objects. Then, if it collides with a wall or brick, it's velocity is reflected. If it collides with a paddle, generate a new velocity based on the position they hit.
