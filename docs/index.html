This project uses SDL2, SDL_mixer, SDL_image, SDL_ttf and rapidjson to implement a game "Breakout". Here is a list of features:

  1. The game has 3 levels. Users have 3 attempts to play this game. To pass a level, clear all bricks on the screen
  2. The game supports multiple languages. Currently it has English version and Chinese version. By editng configuration files, users can change the language used in the game or support new language.
  3. Almost every things can be configured. The types, texture, sound effect, score of bricks can be configured. Users can also define their own maps.
